# Google Play 스토어 등록정보 - 캡처브레인

## 앱 이름
캡처브레인

## 영문 표기
CaptureBrain

## 간단한 설명, 80자 이내
스크린샷을 자동으로 분류하고 Google Drive에 깔끔하게 저장하세요.

## 자세한 설명
캡처브레인은 스크린샷을 찍기만 하면 내용을 분석해 자동으로 정리해 주는 앱입니다.

Google Drive를 연결하고 저장할 폴더 이름만 정하면 준비가 끝납니다. 이후에는 스크린샷을 찍을 때마다 앱이 내용을 감지하고, 적절한 카테고리 폴더를 만들어 이미지와 Markdown 기록을 함께 보관합니다.

주요 기능

- 스크린샷 자동 감지
- 내용 기반 자동 카테고리 분류
- Google Drive 자동 저장
- 스크린샷 원본과 Markdown 파일 분리 보관
- 원하는 루트 폴더명 설정
- 복잡한 대시보드 없이 필요한 설정만 제공

이런 분께 적합합니다

- 업무 자료, 레퍼런스, 영수증, 메모 화면을 자주 캡처하는 분
- 스크린샷 폴더가 금방 지저분해지는 분
- 캡처한 내용을 나중에 검색하거나 정리하고 싶은 분
- Google Drive에 개인 자료를 체계적으로 보관하고 싶은 분

캡처브레인은 사용자가 직접 연결한 Google Drive에 파일을 저장합니다. 앱은 필요한 기능만 제공하고, 스크린샷 정리 과정을 최대한 단순하게 만듭니다.

## 추천 카테고리
- 앱 카테고리: 생산성
- 태그 후보: 생산성, 파일 관리, 자동화, 메모, 도구

## 스토어 이미지 문구
1. 스크린샷만 찍으세요
2. 내용을 보고 자동 분류
3. 이미지와 Markdown을 함께 저장
4. Google Drive에 깔끔하게 보관

## 검색 키워드 후보
스크린샷 정리, 캡처 정리, 구글 드라이브 백업, 자동 분류, Markdown 메모, 화면 캡처 보관, 업무자료 정리
